Asynchrone exemples

Chaque fichier d'exemple est inclu toutes les 1,5 seconde dans le HTML.

Donc chaque fichier est joué sur un intervalle de 1,5 seconde pour gagner en lisibilité.

Chaque fichier reprend le même principe: faire un get sur un json.

Mais chaque fichier comprend une méthode différente.

Ils sont joué dans l'ordre c'est-à-dire

  1. callback.js -> le cas basique ES5
  2. promise.js -> utilisation des promise avec ES2015
  3. asyncAwait.js -> utilisation des mot-clefs async/await avec ES2017
  4. allerplusloin.js -> d'autres cas d'utilisation avec les promises

Maintenant vous pouvez ouvrir la console pour voir les résultats ou vous pouvez recharger la page pour rejouer les scripts (et voir les résultats en live avec la console ouverte).
