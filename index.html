Asynchrone exemples

Each sample file is included every 1.5 seconds in HTML.

So each file is played over a 1.5 second interval to gain readability.

Each file uses the same principle: make a get on a json.

But each file includes a different method.

They are played in the following order:

  1. callback.js -> ES5 callback usage
  2. promise.js -> ES2015 promise usage
  3. asyncAwait.js -> ES2017 async/await usage
  4. gofurther.js -> all & race Promise methods

Now you can open the console to see the results or you can reload the page to replay the scripts          (And see live results with the console open).
